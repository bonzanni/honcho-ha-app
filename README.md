# Honcho Home Assistant App

AI memory infrastructure for Home Assistant. Runs the full [Honcho](https://github.com/plastic-labs/honcho) stack (API server, deriver worker, PostgreSQL with pgvector, Redis) as a single self-contained HA add-on.

## Installation

Add this repository to your Home Assistant add-on store, then install the Honcho add-on.
